<h1>INFO</h1>
This operating system is made for x86 systems, it has its own kernel, own shell,
and libc<br>

<h2>COMPILING</h2>
You must have

```
gcc cross compiler (i686)
binutils cross compiler (i686) 
mtools
grub
qemu
```
Compiling: ` make ` <br>
Running: ` make run ` <br>
Cleaning: ` make clean `
